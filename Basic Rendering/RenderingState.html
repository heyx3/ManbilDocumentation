RenderingState

Stores the state of various rendering options such as which faces to cull.


Metadata

Located in "Rendering/Basic Rendering/RenderingState.h". Has a .cpp file in the same location. Part of the Basic Rendering system.


Declaration

class RenderingState;


Sub-types

  • public enum Cullables;
    • Specifies which surfaces to cull.
    • C_FRONT
      • Cull triangles that face towards the camera.
    • C_BACK
      • Cull triangles that face away from the camera.
    • C_FRONT_AND_BACK
      • Cull all triangles (no triangles will be rendered).
    • C_NONE
      • Don't cull triangles.
  • public enum Cullables;
    • Different ways to ignore pixels of a certain alpha value.
    • AT_NEVER
      • Ignore all pixels, regardless of alpha value.
    • AT_ALWAYS
      • Use all pixels, regardless of alpha value.
    • AT_EQUAL
      • Use all pixels whose alpha is exactly equal to the test value.
    • AT_NOT_EQUAL
      • Use all pixels whose alpha is NOT equal to the test value.
    • AT_LESS
      • Use all pixels whose alpha is less than the test value.
    • AT_GREATER
      • Use all pixels whose alpha is greater than the test value.
    • AT_LESS_OR_EQUAL
      • Use all pixels whose alpha is less than or equal to the test value.
    • AT_GREATER_OR_EQUAL
      • Use all pixels whose alpha is greater than or equal to the test value.

Member Fields

  • public Cullables ToCull;
    • Whether the color buffer should be cleared and replaced with the value of the "ClearedColor" field.
  • public AlphaTests AlphaTest;
    • The test to use when determining whether a pixel should be ignored or not.
    • If a pixel is ignored, or "discarded", its color does not affect the color buffer and it's depth is not written to the depth buffer.
  • public float AlphaTestValue;
    • The value to use for the alpha test.
  • public bool UseDepthTesting;
    • Whether to use depth testing -- if true, parts of triangles will not be rendered if they are behind triangles that have already been rendered.
  • public bool WriteToDepthBuffer;
    • Whether to write triangles to the depth buffer as they are rendered -- if false, triangles will stop obscuring other triangles.

Usage

//Cull triangles that are facing away from the camera
//    like usual.
RenderingState state(RenderingState::Cullables::C_BACK);
state.ToCull = RenderingState::C_BACK;
//If any pixels have less than 0.5 alpha, ignore them
//    and don't let them obscure other triangles
//    behind them.
state.AlphaTest = RenderingState::AT_GREATER;
state.AlphaTestValue = 0.5f;
//Disable depth testing so that triangles will be drawn
//    even if they're being obscured by something else.
state.UseDepthTesting = false;
//Don't let any triangles obscure future triangles
//    that want to use use depth testing.
state.WriteToDepthBuffer = false;

//Now use the state.
state.EnableState();
