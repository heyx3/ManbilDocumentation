Vertices.h

Defines several basic, useful types of vertices. Each one has a basic constructor and a static "GetVertexAttributes()" method that returns a RenderIOAttributes instance representing the vertex data.


Metadata

Has a .cpp file in the same location. Part of the Basic Rendering system.


Vertex Types

  • struct VertexPos;
    • Has a Vector3f "Pos" representing position.
  • struct VertexPosColor;
    • Has a Vector3f "Pos" (representing position) and a Vector3f "Col" (representing color), in that order.
  • struct VertexPosUV;
    • Has a Vector3f "Pos" (representing position) and a Vector2f "UV" (representing texture coordinates), in that order.
  • struct VertexPosNormal;
    • Has a Vector3f "Pos" (representing position) and a Vector3f "Normal" (representing surface normal), in that order.
  • struct VertexPosUVNormal;
    • Has a Vector3f "Pos" (representing position), a Vector2f "UV" (representing texture coordinates), and a Vector3f "Normal" (representing surface normal), in that order.
  • struct VertexPosUVNormalColor;
    • Has a Vector3f "Pos" (representing position), a Vector2f "UV" (representing texture coordinates), a Vector3f "Normal" (representing surface normal), and a Vector3f "Col" (representing color), in that order.
