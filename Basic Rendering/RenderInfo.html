RenderInfo

Represents rendering data that is needed when setting up materials' built-in uniforms.

Member Fields

  • public float TotalElapsedSeconds;
    • The total amount of elapsed time since the game world started.
  • public Camera* Cam;
    • The camera perspective being used to render.
  • public Matrix4f* mView;
    • A reference to the view matrix, which converts from world space to camera space (coordinates are relative to the camera's position and orientation).
    • RenderInfo info(...);
      Camera gameCam(...);

      Matrix4f viewMat;
      gameCam.GetViewTransform(viewMat);
      info.mView = &viewMat;
  • public Matrix4f* mProj;
    • A reference to the projection matrix, which converts from camera space to "NDC" space (a box from {-1, -1, -1} to {1, 1, 1}).
    • RenderInfo info(...);
      Camera gameCam(...);

      Matrix4f projMat;
      //For a 3D perspective view, use this:
      gameCam.GetPerspectiveTransform(projMat);
      //For a 2D orthographic view, use this:
      gameCam.GetOrthoProjection(projMat);

      info.mProj = &projMat;
  • public Matrix4f mVP;
    • The combination of the view and perspective transform; transforms coordinates from world space to NDC space.
    • This field is automatically computed in the RenderInfo constructor.
    • Make sure to set this value if you change the view or projection transform: info.mVP = Matrix4f::Multiply(mProj, mView).
