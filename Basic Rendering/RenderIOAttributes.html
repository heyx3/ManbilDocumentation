    • A single input/output value.

Usage

This class represents the information passed into or out from each shader as a list of "Attributes".

//Create the data type we'll be using for mesh vertices.
struct MyVertex
{
    Vector3f Position, Normal;
    Vector2f TexCoords;
};

//Now define the vertex attributes.
RenderIOAttributes vertData(Attribute(3, false, "vIn_Pos"),
                            Attribute(3, true, "vIn_Normal"),
                            Attribute(2, false, "vIn_UV"));
