Vectors.h

This file contains the various vector classes:

  • Vector2b
  • Vector3b
  • Vector4b
  • Vector2u
  • Vector3u
  • Vector4u
  • Vector2i
  • Vector3i
  • Vector4i
  • Vector2f
  • Vector3f
  • Vector4f

These classes are defined inside the namespace "ManbilVectors" so that they can be differentiated from other libraries' vector classes.

Metadata

Has a .cpp file in the same location. Part of the Lower Math system.


Global Functions

Several functions are defined to cast different vector types to other vector types.
