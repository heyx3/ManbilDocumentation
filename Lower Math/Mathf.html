  • public ClampIntervalResult ClampToIntervalBounds(float intervalStart, float intervalSize, float value)
    • Computes the same thing as ClampToInterval(), but instead of returning the snapped value, this function returns the two values of n that the given value falls between.
    • //In this sample, the interval starts at 0 and is spaced out every 1 unit.
      ClampIntervalResult res = Mathf::ClampToIntervalBounds(0.0f, 1.0f, 0.5f);
      std::cout << res.zero << "\n"; // Outputs 0.
      std::cout << res.one << "\n"; // Outputs 1.

      //In this sample, the interval starts at 2 and is spaced out every 1 unit.
      res = Mathf::ClampToIntervalBounds(2.0f, 1.0f, 0.5f)
      std::cout << res.zero << "\n"; // Outputs -2.
      std::cout << res.one << "\n"; // Outputs -1.

This class is very straightforward; it just contains basic float, double, and int/unsigned int math. It also defines NaN for floats.
