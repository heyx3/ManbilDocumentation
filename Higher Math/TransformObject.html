TransformObject

An object with a position, orientation, and scale. The X axis is left/right, Y axis is forward/back, and Z axis is up/down.

Usage

  • Useful for representing objects in 3D space.
  • When describing this object's rotation with Euler angles, keep in mind that the object is first rotated along the Z axis (yaw), then along the local Y axis (roll), then the local X axis (pitch).
  • At its core, this object uses a Quaternion to represent its rotation, but it also caches its current forward/upward vectors, because these vectors tend to be used often.
