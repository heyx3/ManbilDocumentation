GradientNode

A single value at a specific point in a Gradient.


Metadata

Declaration

template<unsigned int Components>
class GradientNode;
  • "Components" is the number of floats in each gradient value. For example, 4 means the gradient is a Vector4f (or RGBA color).

Usage

  • This class is used by Gradient; each GradientNode represents a single point in the gradient.
//Create a gradient node at the beginning of a gradient.
float bright = 1.0f;
GradientNode<1> start(0.0f, &bright);

//Create a gradient node at the end of a gradient.
float dark = 0.0f;
GradientNode<1> end(1.0f, &dark);
